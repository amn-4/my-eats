# MyEats

a personal app for curating and saving your favourite food spots

## Features

- **User authentication** - sign-in with Clerk
- **Filtering** - filter by suburb, cuisine, dietary requirements, tags, and currently open restaurants
- **Google Places integration** - auto-fetch addresses, phone numbers, and opening hours
- **Dark mode**
- **Responsive design**
- **Easy editing** - add, edit, and delete restaurants

## Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd my-eats
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up the database**
   ```bash
   npm run db:push
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   ```
   http://localhost:3000
   ```
